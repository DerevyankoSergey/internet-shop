# Интернет-магазин (Java)

Консольное приложение.

## Сущности
- `Product` — товар
- `Category` — категория
- `Customer` — покупатель

## Операции
- Поиск товаров по категории
- Фильтрация покупателей по сумме покупок

## Запуск
```bash
javac src/main/java/shop/ui/ConsoleUI.java
java -cp src/main/java shop.ui.ConsoleUI
